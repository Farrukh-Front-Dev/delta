# Interaction Guide: The Dopamine Loop

## The Core Experience

### Task Completion Choreography

**When you click a task checkbox:**

```
0ms    → Checkbox responds instantly (no lag)
50ms   → Task begins smooth fade/scale exit
100ms  → Energy core ring starts filling
200ms  → XP counter begins counting up
300ms  → Celebration overlay appears
500ms  → Glow intensity increases
```

**This creates a satisfying rhythm that makes you want to complete more tasks.**

---

## Component Interactions

### 1. Energy Core
- **Empty (0%):** Dim glow, creates urgency
- **Filling (1-99%):** Glow intensifies, momentum builds
- **Full (100%):** Strong glow, satisfaction achieved

**Try:** Complete tasks and watch the core fill with smooth spring animation.

### 2. Task Items
- **Hover:** Border glows purple, shows interactivity
- **Click checkbox:** Instant response, smooth exit
- **Completed:** Moves to "Completed" section with fade

**Try:** Hover over tasks to see the glow effect.

### 3. XP Counter
- **On completion:** Animates upward with spring physics
- **Smooth counting:** Numbers increment smoothly
- **Visual reward:** Seeing XP increase triggers dopamine

**Try:** Complete multiple tasks quickly to see XP surge.

### 4. Stats Bar
- **XP:** Total experience points earned
- **Level:** Calculated from XP (100 XP per level)
- **Streak:** Daily consistency (future: loss prevention)

**Try:** Hover over stat cards for subtle scale effect.

---

## The Psychology

### Why It Feels Good

1. **Immediate Feedback** - No lag between action and reward
2. **Visual Progress** - Energy core creates completion urge
3. **Smooth Motion** - Spring physics feel organic
4. **Staggered Timing** - Choreography creates rhythm
5. **Ambient Glow** - Subtle reinforcement of achievement

### The Addiction Loop

```
See Task → Anticipate Reward → Complete → Experience Satisfaction → Want More
```

This is intentional. The interaction itself becomes the reward.

---

## Try This

1. **Start with empty core** - Notice the dim, incomplete feeling
2. **Complete one task** - Watch the entire choreography
3. **Complete another** - Feel the momentum building
4. **Fill the core** - Experience the satisfaction of 100%
5. **Add new tasks** - The cycle continues

**The goal: Make discipline feel emotionally rewarding.**
